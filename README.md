# Como rodar o projeto

## Requisitos

- Node.js 14.17.4
- npm 6.14.14
- Angular 12.2

## 📥 Clone o repositório
```bash
git clone https://github.com/mateus-prog/teste-frontend.git
cd teste-frontend
```

## ⚙️ Como instalar
```bash
npm install
```
➡️ Acesse em: `http://server_domain_or_IP:4200`
